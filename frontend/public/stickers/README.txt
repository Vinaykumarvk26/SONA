Place your sticker images in this folder with these exact filenames:

happy.png
sad.png
angry.png
fear.png
disgust.png
surprise.png
neutral.png
idle.png

Notes:
- The result card uses idle.png when no emotion is locked.
- On Retake, the card resets to idle.png immediately.
- If neutral.png is missing, neutral predictions will also fall back to idle behavior if you map it in code.
